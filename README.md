# encheiridion
ἐγχειρίδιον, Greek, means "little thing in the hand".   This is an RTOS client for Perseus' Scaife Library Viewer, which serves a corpus of >3000 classical texts through its XML API. Runs on my Lilygo T3 ESP32S3 dev module.

## features
- search the scaife library (https://scaife.perseus.org/library/) with a touchscreen keyboard to select a text
- read texts and translations in parallel
- savestates with NVS variables
